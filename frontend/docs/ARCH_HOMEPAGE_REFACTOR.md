## 1. БРЕНДИНГ

### 1.1 Логотип
- **Текущее состояние:** Placeholder — div с текстом "LG".
- **Целевое:** Использовать SVG/PNG из `frontend/public/` или `frontend/src/assets/`.
- **Файлы:** `logo.svg` или `logo.png` в `public/`.
- **Компонент:** `Header.tsx` — заменить placeholder на `<img src="/logo.svg" alt="Live Grid" />` или аналогично.
- **Fallback:** Если логотипа нет — оставить компактный текстовый вариант "Live Grid" единым шрифтом.

### 1.2 Кликабельность логотипа
- Логотип уже внутри `<Link to="/">`. Проверить, что клик ведёт на "/".

### 1.3 Favicon и иконки
- **Расположение:** `frontend/public/`
- **Файлы:**
  - `favicon.ico` — 16x16, 32x32
  - `apple-touch-icon.png` — 180x180
  - `icon-192.png` — 192x192 (PWA)
  - `icon-512.png` — 512x512 (PWA)
- **Подключение:** `index.html` — добавить `<link rel="icon">`, `<link rel="apple-touch-icon">` и т.д.
- **Fallback:** Если нет готовых иконок — использовать placeholder.svg как временный favicon.

### 1.4 Типографика "Live Grid"
- **Текущее:** разные размеры между блоками.
- **Целевое:** единый шрифт (Rubik), одинаковый line-height, размеры через Tailwind-классы.
- **Акцент:** "Live Grid." — `text-primary italic`, без скачков размера.

### 4.1 Компонент
- **Расположение:** HeroSection, слева сверху.
- **Отображение:** текущий регион (по умолчанию "Москва и МО").
- **Кликабельность:** по клику — выпадающий список регионов.

### 4.2 Логика
- **localStorage key:** `region`
- **При первом заходе (нет `region`):** показать модал/тост: "Вы находитесь в Москве?" — кнопки "Подтвердить" / "Выбрать регион".
- **Подтвердить:** сохранить "Москва и МО" в localStorage.
- **Выбрать регион:** открыть список (dropdown/modal).
- **Список регионов:** статичный (Москва и МО, СПб, и т.д.) — без гео-детекции.
- **При выборе:** обновить отображение, сохранить в localStorage. Если есть фильтрация по региону — пересчитать.

### 4.3 Компоненты
- `RegionSelector.tsx` — кнопка + dropdown.
- `RegionConfirmModal.tsx` — модал при первом заходе (опционально).
- Или встроить в HeroSection.
